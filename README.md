# Klipper-PLR-Resume
Sistema completo para la recuperación automática y manual de impresiones 3D interrumpidas en Klipper. Utiliza macros de Klipper (.cfg) y un script de shell (.sh) para generar un archivo de recuperación optimizado (re-corte del G-code) basado en las últimas coordenadas (X, Y, Z) guardadas y reinicia la impresión vía Moonraker.
